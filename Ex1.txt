Um Banco de Dados (ou Base de Dados) é uma coleção organizada de dados que se RELACIONAM com algum sentido possibilitando a extração de INFORMAÇÃO. Já um Sistema Gerenciador de Base de Dados (SGBD) pode ser definido como o sistema (software) utilizado para gerir Bases de Dados, permitindo criar, alterar, eliminar e inserir dados na Base de Dados bem como a própria Base de Dados em si.
